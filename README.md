How to install?
``` bash
sudo pacman -S mako waybaar hyprpaper hyprlock fastfetch rofi
yay -S wlogout
git clone https://github.com/wholos/hyprland-dots
Copy files to config
```
Ready!!

How to set mako in default manager in system?

1. Install file setmako.out
2.
``` cpp
./setmako.out
Press Ctrl+C
```
